# External Trial User Briefing Copy - zh - 2026-06-10

Status: draft for controlled trial operators.

Purpose: provide concise user-facing copy for the first controlled Magick AI
Cloud trial. This copy is intentionally narrow. It must not promise automatic
article generation, bulk content production, or direct publishing.

## Short Version

本次试用开放的是 Magick AI Cloud 的托管 AI 辅助能力，用于帮助你整理站内资料、生成写作准备建议、查找相关证据和收集反馈。AI 输出仅作为建议、候选内容或分析结果，不会自动发布，也不会直接修改你的 WordPress 内容。

最终是否采纳、如何编辑、是否发布，仍由你的 WordPress 本地站点和审核流程决定。

## What Is Included

本次试用包含：

- 站内知识检索和证据整理
- 写作准备建议
- 标题、摘要、分类、标签、内部链接等方向的辅助建议
- 可审查的图片候选或媒体辅助建议
- 对 AI 建议结果的反馈提交，用于后续质量评估
- 基础运行状态、错误、用量和支持排查证据

## What Is Not Included

本次试用不包含：

- 自动写完整文章
- 批量生成文章
- 自动发布到 WordPress
- 自动生成或发布假评论、假评价、虚假背书
- 赌博、色情、诈骗、钓鱼、垃圾 SEO、侵权洗稿等用途
- 法律、医疗、金融、安全等高风险建议的自动化生成
- 绕过人工审核、平台规则、AI 标识或内容合规要求

## User Responsibility

请在使用 AI 建议前自行审核：

- 内容是否准确
- 是否适合你的站点和读者
- 是否需要 AI 生成内容标识
- 是否涉及版权、隐私、个人信息或高风险主题
- 是否符合你所在地区的法律和平台规则

AI 输出可能出错，不能替代人工判断、专业意见或最终编辑审核。

## Operational Boundary

Magick AI Cloud 只负责托管运行、检索、建议生成、运行证据和反馈评估。它不是你的 WordPress 发布后台，也不是第二套内容审核系统。

如果你决定采纳某条建议，最终写入、发布和审核仍应通过本地 WordPress / Core 审核链路完成。

## Abuse And Suspension Notice

如果试用站点或请求涉及以下用途，服务方可能拒绝处理、暂停试用或撤销 Cloud API Key：

- 色情或性内容生成
- 赌博、博彩、彩票网站推广
- 诈骗、钓鱼、冒充他人
- 假评论、假评价、虚假社会证明
- 批量垃圾内容、门页、低质量 SEO 站群
- 未授权洗稿、抓取、改写或规避版权
- 规避审核、规避 AI 标识或规避平台检测
- 未经专项审核的法律、医疗、金融、安全等高风险建议

## Operator Script

你可以这样向试用用户说明：

```text
这次试用不是自动写文章或自动发布工具。它主要帮你做站内资料整理、
写作准备、证据检索和建议生成。所有 AI 输出都需要你人工审核，
最终是否写入和发布仍由你的 WordPress 本地流程决定。

我们暂不支持批量文章生成、自动发布、假评论、赌博/色情/诈骗/垃圾 SEO
等高风险用途。如果发现类似用途，Cloud API Key 可能会被暂停或撤销。
```

## Do Not Say

不要使用这些说法：

- "自动帮你写完整文章并发布"
- "批量生产 SEO 文章"
- "无需人工审核"
- "自动涨流量"
- "自动生成评论和评价"
- "绕过平台检测"
- "不限用途"

## Safe Copy

推荐使用这些说法：

- "可审查的 AI 建议"
- "写作准备辅助"
- "站内知识证据整理"
- "Cloud 托管运行能力"
- "本地审核和发布仍由你控制"
- "反馈用于改进建议质量，不会自动改变生产行为"
